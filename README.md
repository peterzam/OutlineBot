# OutlineBot
## Custom discord bot for GetO2MM.
<hr>

## 1. How to run :

1. Copy and add tokens of Dockerfile

2. Docker Build
``` bash 
docker build -t peterzam/outlinebot .
```
3. Docker run
``` bash
docker run -d --name outlinebot peterzam/outlinebot
```
<hr>

## 2. Commands : !req, !request, !vpn, !about
